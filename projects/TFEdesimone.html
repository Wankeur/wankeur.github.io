Final year project in automation

Completed: June 2025 Duration: 4 months
Siemens TIA Portal HMI WinCC Unified

Project Overview

This project involved the complete programming of a special machine intended for a research and development center. It was carried out as part of my final year internship in automation. This machine is designed to create prototypes of insulating panels.

Key Features

  • Complete management of manufacturing parameters
  • Revamping of an old hydraulic press
  • Production data logging and analytics
  • Enterprise standards applications
  • Position and temperature regulation

Technical Implementation

The system was built using Siemens S7-1500 PLCs with TIA Portal for programming and configuration. The HMI interface was developed using WinCC for operator interaction and monitoring.

Architecture

  • Control system with a single PLC
  • Industrial Ethernet network for communication
  • Integration of a Siemens and a Festo drive communicating in PROFINET
  • Servos motors for the deplacements

Challenges and Solutions

Challenge: Precise Regulation with an Old Press

The old press was not designed for the precision required by the client.

Solution: Implemented a new control algorithm and two different ascent speeds to improve precision.
